Twitch Installs Arch Linux

Remember how chaotic Twitch Plays Pokemon was? Now we have a much harder challenge: install Arch Linux. Every five seconds, the most popular keystroke in Twitch chat will be entered into an Arch Linux virtual machine.

Up for the challenge?

Bonus Points!

Twitch Installs Arch Linux has no rules (except don't do anything illegal).

However, below are some milestones. When these goals are met, we will raffle off some cool swag!

  • Boot Arch Linux from the hard disk
  • Write a python 'Hello World!' script
  • Configure a fully working X server
  • Pull up the Twitch screen in the virtual machine!
  • Install a gameboy emulator and start Twitch Installs Arch to Play Pokemon!
  • Install irssi and join the #twitchinstalls coldfront.net chat!
