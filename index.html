Twitch Installs Arch Linux

This is the journey to install a Linux distro through Twitch chat. Every ten seconds, the most typed key in our Twitch channel will be input to our Arch virtual machine. Having to work with all of the other viewers' input adds to the already existing challenge of installing Arch.

Get Involved!

Just join our Twitch.tv stream and type as many key presses in chat as your heart desires!

End Game

In order for Arch to be considered "beaten" Arch and X server need to be properly configured, so that the viewers can launch a webpage in the Arch instance that will signal the end of the experience.
